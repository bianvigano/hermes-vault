# Foam Graph

Visualisasi knowledge graph dari wiki notes.

## Cara Buka
1. Buka folder `~/.hermes/vault/` di VS Code
2. Install Foam extension (foam.foam-vscode)
3. Command palette → "Foam: Show Graph"

## Link Format
- `[[nama-file]]` — WikiLink ke note lain
- Tag: `#tag` untuk kategorisasi

## Graph Legend
- Node = note/file
- Edge = link antar note
- Orange = active/selected node
- Blue = regular note
- Gray = placeholder (link ke file yang belum ada)
